# Shopping Cat 

Adopt, Don't Shop! 請以領養代替購買！

## 使用技術

- [Deno](https://deno.land/)
- [TypeScript](https://www.typescriptlang.org/)
- [Fresh](https://fresh.deno.dev/)

## 啟動

- 安裝 `deno`
- 在本機執行 `deno task start` 啟動測試環境
- 執行 `deno test` 可執行測試
- 打開瀏覽器 `http://localhost:8000/`
